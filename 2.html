Yoongrammer's POE GUIDE

  1. 아이템을 직접 거래할 수 있는 poe.trade를 접속해보자!

    2. ① 화폐시장 : 화폐(오브)의 물물교환비(시세)를 알아 볼 수 있는 페이지
    ② 리그선택 : 거래할 리그 선택
    ③ 아이템 타입(대분류) : 장비, 보석, 장신구 등등 어떤것을 검색할지를 분류하는 필터
    ④ 아이템 타입(소분류) : 대분류에서 분류한 장비의 세세한 타입을 검색하는 필터
    ⑤ 이름으로 검색 : 장비의 고유 이름을 직접 검색해서 찾는 방법
    ⑥ 장비옵션 검색 : 장비에 들어갔으면 하는 옵션을 직접 선택해서 해당옵션의 장비만 표시해주는 필터
    ⑦ 장비옵션 추가 : 클릭 시 추가로 장비옵션 선택 가능


  2. 여러 solt 기능을 사용하여 원하는 아이템을 검색하였다면, 자동으로 게임 속 귓속말을 할 수 있게 해준다.

    3. ① 장비 : 장비의 종류와 옵션 등 스펙
    ② 가격 : 장비의 구매가격, 구매에 필요한 화폐의 종류와 갯수가 표시
    ③ 접속 상태 : 판매자의 접속상태를 표시. Online(접속 중) Offline(접속 안함)
    ④ 판매자 이름 : 아이템을 판매하고있는 판매자의 ID를 표시
    ⑤ 귓속말 : 판매자에게 구매 귓속말을 보내는 기능. Whisper를 클릭 시 구매에 필요한 메세지를 복사하게 되고,
    ⑥번처럼 Copied to clipboard라는 메세지로 변경됨. 이후 게임내에서 채팅창에 Ctrl+V를 누르고 엔터를 누르면 자동으로 판매자에게 해당 아이템을 구매한다는 메세지를 전달하게됨.
    이후 보통 10초내에 답변이 오고 파티초대를 해서 거래가 가능. 반드시 구매에 필요한 화폐를 준비한 상태로 귓속말을 보내자.








  3. 여기서 하이드 아웃이란, 자기 계정의 my home과 같은 맥락이다. 구매하는 경우는 visit hideout을 하여 거래를 하고, 판매하는 경우에는 구매자가 알아서 찾아온다.
